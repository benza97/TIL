# 원격저장소 정리
##  저장소 처음 만들때 
- `git init`

## 버전 기록할 때
- `git add`
- `git commit -m`

## 상태 확인할 때
- `git status` 1통,2통
- `git log`  커밋 확인
---

# 원격 저장소 활용하기
## 처음 연결할 때
- `git remote add origin URL`

## 보내기/받기
- `git push origin master` 보내기
- `git pull origin master` 받기

## 원격 프로젝트 참여할 때
- `git clone url`

## 원격 저장소 삭제
- `git remote rm`

## 원격 저장소 정보확인
- `git remote -v`

---

# 용어 
## 브랜치 `master,main`
## 원격저장소 `origin`
